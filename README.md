# Web automation test on OrangeHRM

This is a class project on Web Automation Course in Alterra Academy. 
This project is built on Maven with Java.

Dependencies used:
- Serenity BDD
- Cucumber
- Serenity Screenplay
- Serenity Webdriver
- Serenity Ensure
- DotEnv

**Webdriver used is Chromewebdriver v100**
